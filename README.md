# OCaml Learning Repository

本仓库用于 **学习 OCaml 语言**，内容主要包括一些 **小型 project、练习（exercises）以及独立的 `.ml` 文件**，用于熟悉 OCaml 的语法、类型系统和函数式编程思想。

## 内容说明

仓库中的代码主要涵盖以下几类内容：

- 📌 **基础语法练习**
  - 基本数据类型
  - 模式匹配（pattern matching）
  - 函数定义与递归
  - 高阶函数

- 📌 **小型练习（Exercises）**
  - 列表与递归相关练习
  - 常见函数式编程问题
  - 简单算法实现

- 📌 **小项目（Mini Projects）**
  - 用于综合练习 OCaml 的小程序
  - 侧重类型设计与函数式风格

- 📌 **独立 `.ml` 文件**
  - 一些零散但有代表性的示例代码
  - 用于测试或实验语言特性

## 目标

- 熟悉 **OCaml 的核心语法与类型系统**
- 理解 **函数式编程的基本思想**
- 了解**MirageOS**等`Unikernel`
- 了解如何实现系统的形式化验证
